# DEC-004: Dam Pump Concept

## Status

Accepted

## Date

2026-07-04

## Decision

Proceed with concept planning for a solar-only dam recirculation pump using second-hand panels, targeting approximately 6,000 L/hour at around 20 m lift, with creek/cascade return preferred over fountain.

## Context

The dam pump is a non-critical load and does not need to depend on the main workshop battery system. A solar-only system may provide useful daytime water movement without increasing grid or battery demand.

The current concept includes approximately 3-5 kW of panels, a 1.1-1.5 kW pump, and a lower plunge basin or waterhole concept.

## Rationale

- Solar-only operation is appropriate for a non-critical system.
- Second-hand panels may be cost-effective where variable output is acceptable.
- A creek/cascade return may provide better amenity, aeration, and erosion-control design than a fountain.
- Keeping the dam pump separate protects the main energy system from non-critical load growth.

## Consequences

- Lift height and pipe run need to be measured.
- Pump sizing advice is required.
- A sunny panel location must be identified.
- Plunge basin, spillway, waterway, and earthworks approval risks must be checked.
- Electrical safety and compliance still apply even with second-hand panels.
