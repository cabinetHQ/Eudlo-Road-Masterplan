# Changelog

## Version 1

Initial governance baseline for Project Eudlo.

### Project Zero Architecture Transition

- Defined Project Zero as the Eudlo App and a separate property operating system platform.
- Created Project Zero vision, roadmap, architecture, backlog, and repository review.
- Added DEC-007 for Project Zero architecture direction.
- Identified that the repository must evolve from document-first knowledge base into entity, database, GIS, workflow, automation, and AI architecture.
- Added canonical data model, entity relationship model, database schema draft, GIS data model, migration plan, and DEC-008.
- Established structured entities as the future source of truth while keeping Markdown as narrative and decision archive.

### Completed So Far

- Created the base Project Eudlo repository structure.
- Established [README](../README.md) and [Master Index](../docs/00_Index/Master_Index.md) as primary navigation points.
- Created energy infrastructure planning for 50 kW solar + 100 kWh battery with future 100 kW solar expansion.
- Created financial strategy and Stage 1 solar/battery business case.
- Created financial model output and Python model for savings, payback, finance, and offset comparisons.
- Created grants, incentives, finance, and R&D strategy documents.
- Created water, dam, irrigation, and dam solar pump concept documents.
- Created decision records DEC-001 through DEC-006.
- Created action register and structured action CSV.
- Created initial asset register and structured `assets.csv`.
- Created Digital Twin strategy and asset-centric app architecture direction.
- Created governance framework for future contributors and AI agents.

### Current Key Decisions

- DEC-001: Design infrastructure for 100 kW solar expansion but initially model 50 kW solar + 100 kWh battery.
- DEC-002: Use the current $1,200/month electricity bill as the primary financial baseline.
- DEC-003: Stay grid-connected initially while designing for microgrid capability.
- DEC-004: Proceed with concept planning for a solar-only dam recirculation pump.
- DEC-005: Treat solar/battery incentives separately from workshop efficiency grants.
- DEC-006: Use an asset-centric digital twin architecture.
